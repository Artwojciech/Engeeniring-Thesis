Celem pracy inżynierskiej jest wykonanie profesjonalnego i nowoczesnego portfolio fotograficznego dostępnego jako aplikacja webowa. Będzie to reklama działalności fotografa, pozwalająca odwiedzającym stronę na zapoznanie się z jego pracami i uzyskanie informacji o nim samym. Aplikacja umożliwia przeglądanie zdjęć podzielonych na kategorie tematyczne. Dodatkowo zalogowani użytkownicy otrzymują możliwość utworzenia listy ulubionych zdjęć. Administrator systemu ma możliwość zarządzania galeriami zdjęć i dodawania do nich nowych zdjęć z poziomu aplikacji. Wszystkie dane przechowywane będą w relacyjnej bazie danych. Do stworzenia interfejsu użytkownika wykorzystana zostanie biblioteka React, a zaplecze serwerowe oparte będzie na frameworku Express.js.
